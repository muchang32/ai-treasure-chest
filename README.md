# AI精選寶箱

精選 AI 工具目錄，幫助你快速找到提升工作效率與創意的 AI 工具。

## 功能

- 依分類瀏覽 AI 工具
- 關鍵字搜尋
- 工具卡片含網站截圖預覽
- 支援深色模式與字體大小調整
- 手機版響應式設計

## 技術架構

- Vite + React + TypeScript
- Tailwind CSS + shadcn/ui
- TanStack Query
- 資料來源：Google Sheets（CSV 匯出）

## 本地開發

```bash
npm install
npm run dev
```

## 部署

推送到 `main` 分支後，GitHub Actions 會自動 build 並部署至 GitHub Pages。
